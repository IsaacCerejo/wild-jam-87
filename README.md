# Fungal Protocol

**Fungal Protocol** is a game developed for **Godot Wild Jam 87**.  

You can play it on [itch.io](https://aaatum-games.itch.io/fungal-protocol).

---

## Description

**MUSHROOMS GROWING ON PEOPLE???**

Save these poor humans before they DECAY and succumb to the mushroom zombie virus!

**Controls:**

| Action       | Key / Mouse |
|-------------|------------|
| Grab tools / Use | Left Mouse Click |
| Fullscreen       | F |
| Pause            | Tab |
